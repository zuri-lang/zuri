# Cleaning cache and log files

- To clean all cache and log file, run the command `zuri clean`.
- To clean log files only, pass the `--logs` flag (or `-l` for short). For example, `zuri clean --logs`
- To clean cache files only, pass the `--cache` flag (or `-c` for short). For example, `zuri clean --cache`.
- You can pass the `--all` flag (or `-a` for short) to clean everything. This is the same as simply running `zuri clean`.
